# expense-tracker-ui
Frontend UI for the Expense Tracker application built with Angular 19 and Tailwind CSS.

Frontend application for the Expense Tracker system.

## Tech Stack
- Angular 19
- Tailwind CSS
- TypeScript

## Features
- Expense dashboard
- Add / Edit expense
- Expense analytics
- Responsive UI

## Installation

1. Clone the repository
2. Install dependencies

npm install

3. Run the project

ng serve

4. Open browser

http://localhost:4200
